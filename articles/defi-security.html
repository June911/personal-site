Crypto · First Principles

开放性
的代价

区块链的价值和风险来自同一个东西——开放性。这不是 bug，是定义。DeFi 安全的第一性原理推导。

金融的本质问题

金融 = 不完全信任的人之间转移价值。所有金融系统存在的原因，都是降低信任成本。

传统金融的方案是用制度降低信任成本：银行、法院、监管、央行。代价是制度租金和准入门槛。全球约 13 亿成年人没有银行账户，跨境汇款平均被中间人吃掉 6-7%。2008 年证明制度信任本身也会系统性失败——没人知道 MBS 里装了什么。

区块链提出了另一种方案：用协议属性降低信任成本，不依赖特定机构。三个属性——

属性含义当前实现
可验证性不用信任某个人，自己能验证源码公开 + 链上状态公开
无许可不用获得机构批准才能参与任何地址都能发交易
承诺可信规则自动执行，单方不能改智能合约 + 治理/分叉的显性成本

这三个属性是区块链相对传统金融的全部差异化价值。拿掉任何一个，区块链就退化为一个效率更低的数据库。

价值来源 = 风险来源

每一个降低诚实参与者摩擦的属性，同时降低了攻击者的摩擦。
属性对诚实方对攻击方
可验证性能审计规则、验证执行能分析所有逻辑、构造精确 exploit
无许可全球任何人都能参与攻击者不需要许可，无法被阻止
承诺可信规则可信，可以组合构建Bug 修复慢，需共识同意才能改

这不是区块链的 bug，是所有开放系统的根本属性。互联网、开源软件、自由市场——全部面对同一个张力。区别在于后果：互联网泄数据可以打补丁，区块链上交易一旦执行，没有东西可以修复。

2026 年 4 月 Drift Protocol 被攻击，损失 2.85 亿美元。攻击者没有利用代码漏洞，而是通过社会工程获取多签权限，12 分钟内系统性抽走资产。链上你能看到资金流向，但你不能阻止。"无需许可"是 crypto 的核心价值，同一个特性让被盗资金无法追回。

传统金融不是"更安全"

它有一套损失社会化机制——存款保险、央行兜底、法律追偿。你感觉不到被盗，因为有人替你承担了。DeFi 把这层缓冲去掉了，所有风险直接暴露给用户。

智能像水

AI 对 DeFi 安全的影响，本质上不是让某一种攻击变强了，而是——智能像水一样，同时灌入所有缝隙。

水的特点不只是往低处流，是找到所有路径。你堵住一条，水从另一条走。AI 让所有可能的攻击路径同时被搜索，成本趋近于零。2025 年全年 crypto 行业被盗 34 亿美元，仅上半年就突破 22 亿（Chainalysis）。2026 年 AI 已经能在操作系统中发现数十年未被发现的高危漏洞并自主构造 exploit——智能合约的复杂度远低于操作系统，且全部公开。

但"攻击更强"只是表象。更深的根因是：AI 放大了攻击者天然的组织效率优势。

攻击者防御者
决策闭环秒级（单人 / 小团队）月级（DAO 治理 / 多方协调）
AI 采用摩擦零（下载模型，扫描合约）高（治理批准，集成协议）
激励利润全归自己安全是公共品，难以 justify 预算

竞争不是攻击技术 vs 防御技术，是攻击者的决策速度 vs 防御者的决策速度。水是瞬时的，坝的建造受限于治理效率。

不可解的三元约束

所有防御手段本质上都是在开放性中加回摩擦。更精确地说——

无许可 / 快速响应 / 去中心化治理——最多同时满足两个。
  • 无许可 + 快速响应 → 需要中心化暂停权（牺牲去中心化治理）
  • 无许可 + 去中心化治理 → 治理天然慢，无法秒级响应
  • 快速响应 + 去中心化治理 → 需要预设规则限制触发者（引入许可门槛）

每一种防御都在削弱区块链的差异化价值。加够多的摩擦，DeFi 就变成了跑在区块链上的传统金融——更慢、更贵、没有制度兜底。DeFi 的风险不会被"解决"，只会被重新分配。

六条正反馈回路

核心不等式（诚实方收益 > 攻击方损害）一旦被市场感知为恶化，六条自我强化的回路同时启动：

资本螺旋攻击 → TVL↓ → 协议收入↓ → Token↓ → 安全预算↓ → 防御↓ → 攻击更容易
结构陷阱小协议被淘汰 → 资金集中头部 → 表面更稳 → 单点系统性更强
信任传染某协议被攻 → 用户怀疑所有链上交互 → 自托管退潮 → 托管化
制度包围事故↑ → 保险退出 / 银行收紧 / 监管重构 → 合规成本↑ → 空间收缩
创新窒息安全成本↑ → 开发者流失 → 新原语消失 → 增长叙事弱 → 资本↓
跨行业绞杀DeFi 事故 → AI 被污名化 → 开源模型受限 → 防守工具更弱 → 更多事故

单条可消化，多条同时触发 = 系统性事件。

但 DeFi 不会消失

因为传统金融的问题也是真实的。13 亿人被排除在银行体系之外，跨境汇款被吃 6-7%，金融产品定价是黑箱。

三个不可逆的力量

  • 需求不依赖于特定实体 — FTX 倒了 Binance 接住，Drift 被黑资金流向 Aave。供给侧个体失败只是份额重分配
  • 透明是棘轮 — 见过链上透明度后，无法假装传统金融的不透明是合理的
  • 中间人必然寻租 — 激励结构决定的。只要有寻租就有绕过中间人的经济动力

TVL 从 2023 年底部 378 亿恢复到约 945 亿，但远未回到 2021 年的 1775 亿峰值。DeFi 没有死，但也没有爆发——核心需求真实，规模比叙事暗示的小。

四层可解性

层级性质可解性
代码层工程问题可解（形式化验证、ZK、AI 防御）
单协议系统层复杂性问题可缓解（断路器、限速、隔离）
跨协议交互层组合爆炸问题不可预见但可兜底（containment）
数学层逻辑性质不可解（结构定义）

跨协议交互层

这是最容易被忽略的一层。N 个可组合协议的交互方式是指数级的，不可能逐一审计。2022 年 Terra → 3AC → Celsius 连环清算没有"攻击者"，是系统耦合的必然结果——Normal Accident。

但"不可预见"不意味着"不可处理"。核电行业的方案不是消除所有可能的故障交互，而是让每种故障的后果都被安全壳（containment）兜住。不需要审计所有组合，只需要每种组合的损失有上界。

数学层：三个不可解

  1. 价值来源 = 风险来源 — 开放系统的定义
  2. 三元约束最多满足两个 — 逻辑证明
  3. 治理公地悲剧 — 安全是公共品。Aave 用 150 万安全预算保护 260 亿链上资产（占 TVL 0.006%，占年收入约 1%）。博弈论结构

建坝

智能像水，但水可以建坝。坝有两种。

技术坝

三条路径（不互斥，可分层组合）

  • 路径 A：ZK 化 — 结算逻辑公开可证明，策略逻辑 ZK 包起来。验证结果但看不到过程。牺牲完全可验证性
  • 路径 B：损失上界 — 承认会被攻破，靠限速/隔离/限额控制每次损失。牺牲资本效率
  • 路径 C：速度防御 — AI 防御在同一区块内拦截异常。牺牲：防御系统成为新的信任依赖

制度坝

历史上最有效的坝不是技术的。银行体系的安全靠 Basel 协议强制资本充足率 + 存款保险 + 央行最后贷款人。

DeFi 需要自己的 Basel——不可治理变更的安全底线标准，用标准准入替代强制执行。部署了限速标准的协议可以被保险覆盖，没部署的自担风险。不是强制，是激励兼容。

谁付钱建坝？

基础设施寡头有激励补贴安全——不是利他，是安全就是护城河。Cloudflare 提供免费 DDoS 防御的逻辑：用安全换垄断。但寡头补贴有前提：寡头本身不能脆弱。Token 暴跌 → 安全预算归零 → 坝塌。

终局

窗口期的三条轨迹

A. 渐进强化

蓝筹持续安全运行，标准提高，ZK/AI 防御成熟

概率：中

B. 僵局

不死不活，传统金融缓慢吸收技术，纯 DeFi 被边缘化。不是崩溃，是被忽略

概率：可能最高

C. 系统性事件

级联崩溃触发赛道级分叉。DeFi 的切尔诺贝利

概率：低，一旦发生不可逆

终局结构

应用层（KYC/保险/客服）→ 中间层（评级/保险协议）→ 协议层（TCP/IP 化的少数存活者）

三种融合形态

  • 制度引导的融合（最优）— 设定安全底线标准，保留无许可创新空间
  • 制度收编的融合（次优）— 传统金融选择性采用效率，丢弃可及性。Circle 冻结权限、BUIDL 仅合格投资者可能是这条路径的早期信号
  • 制度消灭后的伪融合（最差）— DeFi 被定义为非法，区块链退化为后端数据库

DeFi 的"胜利"不会以取代传统金融的形式出现，而是传统金融不知不觉变成了 DeFi——底层链上结算，上层制度保护。就像现在用手机转账不会想到背后有 TCP/IP。

我的判断

反脆弱层级悖论

行业是反脆弱的，个体是脆弱的。行业通过杀死个体来变强，个体是脆弱的消耗品。Drift 的"达尔文淘汰"对行业是进化，对 Drift 的用户是灾难。

你可以相信行业是反脆弱的，同时不想成为让它变强的那个牺牲品。这不是矛盾，是理性。

确定的

  • 价值来源 = 风险来源，结构性的
  • AI 放大攻击者组织效率优势
  • 三元约束不可绕过
  • 终态是融合，形态未定

不确定的

  • 不等式是否已翻转
  • AI 防守能否缩小差距
  • ZK 能否解耦可验证性与透明性
  • 融合对谁有利

短期不参与 DeFi 挖矿。不是因为 DeFi 会死——恰恰因为它不会死，它会通过让一些参与者受损来变得更强。我不想做那个参与者。

长期仍然认同区块链的价值。重返链上的信号不是"技术更先进了"，是"有人被攻击了但防住了"——防御体系的实战验证。这个信号出现之前，观察比参与更明智。
